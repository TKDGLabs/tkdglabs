# TKDG Labs 검색 등록 체크리스트 (GSC + 네이버)

이 문서는 `https://tkdglabs.com/` 기준 제출 순서입니다.

## 1) 사전 확인
- [ ] 사이트가 HTTPS로 정상 열림
- [ ] 메인 URL: `https://tkdglabs.com/`
- [ ] 사이트맵 URL: `https://tkdglabs.com/sitemap.xml`
- [ ] robots URL: `https://tkdglabs.com/robots.txt`
- [ ] LLM 안내 URL: `https://tkdglabs.com/llms.txt`

## 2) Google Search Console 등록
- [ ] [Google Search Console](https://search.google.com/search-console) 접속
- [ ] 속성 추가: `URL 접두어` 방식으로 `https://tkdglabs.com/` 등록
- [ ] 권장: `HTML 파일 업로드` 또는 `DNS TXT`로 소유권 인증
- [ ] `색인 > Sitemaps`에서 `sitemap.xml` 제출
- [ ] `URL 검사`로 홈 URL 인덱싱 요청
- [ ] 24~72시간 후 `페이지 색인` 상태 재확인

## 3) 네이버 서치어드바이저 등록
- [ ] [네이버 서치어드바이저](https://searchadvisor.naver.com/) 접속
- [ ] 사이트 등록: `https://tkdglabs.com/`
- [ ] 소유 확인(HTML 태그 또는 파일 업로드)
- [ ] `요청 > 사이트맵 제출`에서 `https://tkdglabs.com/sitemap.xml` 제출
- [ ] `요청 > RSS 제출`은 블로그/뉴스 발행 시작 후 진행

## 4) 인덱싱 체감 가속 팁
- [ ] 홈 본문에 핵심 키워드 유지: 브랜드 런칭, 콘텐츠 마케팅, 비주얼 프로덕션
- [ ] FAQ 섹션 유지/확장 (질문형 검색 대응)
- [ ] 신규 실적/사례 업데이트 시 본문 문장 변경 후 재요청
- [ ] 외부 링크 확보: 사람인, 브랜드 사이트, 협력사 언급 페이지

## 5) 검수용 검색 쿼리
- `site:tkdglabs.com`
- `tkdglabs 브랜드 런칭`
- `티케이디지랩스 콘텐츠 마케팅`
- `턴키하우스 by TKDG`
